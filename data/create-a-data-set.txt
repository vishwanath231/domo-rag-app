Create a DataSet
POST
https://api.domo.com
/v1/datasets
Creates a new DataSet in your Domo instance. Once a DataSet has been created, data can be imported into the DataSet.
Request
Security: Bearer Auth
Body
application/json
name
string
required
Name of the DataSet to create
description
string
Description of DataSet to create
schema
object
required
The current schema associated with this DataSet
columns
array[object]
required
Array of columns in the DataSet
Responses
201
Returns a DataSet object when successful. The returned object will have DataSet attributes based on the information provided when the DataSet was created.
