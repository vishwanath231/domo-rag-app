Delete a page
DELETE
https://api.domo.com
/v1/pages/{page_id}
Permanently deletes a page from your Domo instance.
Warning
This is destructive and cannot be reversed.
Request
Security: Bearer Auth
Path Parameters
page_id
string
required
The ID of the page
Example:
876655018
Responses
204
Returns the parameter of success or error based on the page ID being valid.
Auth
Token
:
Parameters
page_id*

Send API Request
Request Sample: Shell / cURL
curl --request DELETE \
  --url https://api.domo.com/v1/pages/page_id \
  --header 'Authorization: Bearer undefined'
