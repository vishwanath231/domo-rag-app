Update a Personalized Data Permission (PDP) policy
PUT
https://api.domo.com
/v1/datasets/{dataset_id}/policies/{pdp_id}
Update the specific PDP policy for a DataSet by providing values to parameters passed.
The number of characters for the list of values for a single PDP policy, including a delimiter character for each value, must be less than 255 characters.
Request
Security: Bearer Auth
Path Parameters
dataset_id
string
required
pdp_id
string
required
Body
application/json
Name of the Policy
name
string
filters
array[object]
column
string
Name of the column to filter on
values
array[string]
Values to filter on
operator
string
Matching operator (EQUALS)
not
boolean
Determines if NOT is applied to the filter operation
type
string
Type of policy (user or system)
users
array[integer]
List of user IDs the policy applies to
groups
array[integer]
List of group IDs the policy applies to
Responses
200
Returns a subset of the DataSet object specific to the data permission policy.
