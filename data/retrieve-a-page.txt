Retrieve a page
GET
https://api.domo.com
/v1/pages/{page_id}
Retrieves the details of an existing page.
Request
Security: Bearer Auth
Path Parameters
page_id
string
required
The ID of the page
Example:
876655018
Responses
200
Returns a page object if valid page ID was provided.

application/json
