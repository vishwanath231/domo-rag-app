Create a page
POST
https://api.domo.com
/v1/pages
Creates a new page in your Domo instance.
Request
Security: Bearer Auth
Body
application/json
name
string
The name of the page
parentId
integer
If provided, the page will be created as a subpage to the page provided
locked
boolean
will restrict other users the ability to make edits to page or its content - the default value is false
cardIds
array[integer]
The IDs of all cards to be added to the page
visibility
object
Determines the access given to both individual users or groups within Domo
userIds
array[integer]
The IDs of users that will be given access to view the page
groupIds
array[integer]
The IDs of groups that will be given access to view the page
Responses
200
Returns a page object when successful.
